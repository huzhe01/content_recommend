# 现金回赠与生活类

## 现金回赠最重要的四个变量

- **类别加成**：餐饮/超市/网购/交通/海外
- **上限**：每月/每季/每年
- **计算口径**：是否含电子钱包、是否含缴费、是否含分期
- **兑现方式**：直接抵扣/积分再换/需要兑换/最低兑换额

- 交通：____（地铁/的士/网约车/加油）

## 组合策略

- **主力通用回赠卡**：覆盖“其他类别”
- **高加成场景卡**：只用在餐饮/网购等高频类别
- **上限管理**：每月先刷高加成到上限，再切回通用卡
