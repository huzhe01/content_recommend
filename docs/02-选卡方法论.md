# 选卡方法论

## 先定义“胜利条件”

请先选一个主目标（只选一个）：

- A. **现金回赠最大化**（越稳定越好）
- B. **里数/积分最大化**（愿意做任务、懂兑换）
- C. **旅行体验最大化**（贵宾室/保险/酒店会籍/升级）
- D. **省心**（少卡、少规则、少踩雷）

## 计算规则（建议统一口径）

- **欢迎礼回报率**：欢迎礼价值 ÷ 首期任务刷卡金额
- **长期回报率**：一年（预期回赠价值 − 年费 − 额外成本）÷ 一年刷卡金额
- **上限处理**：把“类别上限/季度上限/每月上限”折算成年度可拿到的最大回报

## 典型策略（占位符）

- **新手省心组合**：主力卡（通用）+ 补充卡（餐饮/网购）
- **里数玩家组合**：主力里数卡 + 里数加速卡 + 旅行权益卡
- **外币/旅行多**：外币回赠/免外币手续费 + 旅行保险/贵宾室

## 常见坑（先占坑，后续补细）

- 欢迎礼**不算某些交易**（例如：电子钱包增值、税费、缴费平台等）
- 回赠类别**定义模糊**（MCC 归类差异导致拿不到加成）
- 里数/积分**过期规则**与兑换限制
- 年费减免**需要主动申请**或有隐藏门槛
