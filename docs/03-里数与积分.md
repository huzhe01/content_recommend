# 里数与积分

## 先搞清楚：你到底在攒什么

- **银行积分**：可换现金回赠/礼品/里数（通常有兑换比例）
- **航空里数**：可换机票/升舱/奖励票（价值弹性大）
- **酒店积分**：可换房晚/升级（同样弹性大）

## 价值评估（模板）

> 你可以把“每 1 里数/积分值多少钱（HKD）”写成自己的假设。

- 你的里数估值：____ HKD / 里
- 你的酒店点数估值：____ HKD / 点
- 你的积分兑换损耗（手续费/折价/税费）：____

## 兑换路径（占位符）

- 银行积分 → 航空里数：____（比例、最小兑换、到账时间）
- 银行积分 → 酒店积分：____
- 里数最佳用途：____（短途/长途/商务舱/升舱/伙伴票）

## 计算示例（留空）

- 场景 1：每月餐饮 ____ HKD，能攒 ____ 里，等价回报率 ____%
- 场景 2：每年旅行 ____ 次，兑换 ____，总价值约 ____ HKD

## 注意事项

- **过期**：里数/积分是否滚动延期？是否要有活跃记录？
- **税费/燃油附加费**：奖励票不等于“0 成本”
- **兑换可用性**：旺季/热门航线可能很难换
