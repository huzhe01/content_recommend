# 香港攻略信用卡

> 一站式「香港信用卡」攻略大纲：选卡方法、权益拆解、里数/积分、申请与用卡避坑。你可以在每个章节直接补充你自己的经验与数据。

!!! warning "免责声明"
    本站内容仅作信息整理与经验分享，不构成任何投资/理财/税务/法律建议。信用卡条款与优惠可能随时调整，请以银行/发卡机构最新公告为准。

## 你可以从哪里开始

- **只想快速选一两张主力卡**：先看「快速开始」→「选卡方法论」。
- **追里数/积分最大化**：看「里数与积分」与「旅行与机场权益」。
- **日常消费回赠为主**：看「现金回赠与生活类」。
- **准备申请/总被拒**：看「申请流程与材料」与「风控/封卡/拒批应对」。

## 本站结构（可随时扩写）

- **快速开始**：用 10 分钟明确你的需求与边界条件
- **选卡方法论**：把“欢迎礼 + 长期回赠 + 费用/门槛 + 兑换成本”放到同一张表里比
- **权益与回赠**：里数/积分、旅行权益、现金回赠
- **申请与用卡**：申请材料、批卡逻辑、年费与费用、风控与拒批
- **附录**：术语表、FAQ、更新日志

## 你后续补充内容的推荐格式

每个章节建议都包含以下模块（按需选用）：

- **适用人群**：谁最适合/谁不适合
- **关键结论（TL;DR）**：3-5 条
- **参数表**：年费、门槛、回赠上限、外币手续费、兑换比例等
- **计算示例**：用 2-3 个真实消费场景算回报
- **坑与注意事项**：条款、限制、叠加规则
- **版本记录**：什么时候更新过，改了什么
