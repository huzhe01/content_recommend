# 申请流程与材料

## 申请前自检（模板）

- 你申请的是：____（香港本地银行/国际发卡组织/联名卡）
- 你希望拿到的信用额度：____
- 你能提供的入息/资产证明：____
- 你是否有本地信用记录：____（有/没有/很少）

## 常见材料清单（占位符）

- 身份证明：____
- 地址证明：____（近 ___ 个月）
- 入息证明：____（受雇/自雇不同）
- 银行月结单/存款证明（如需要）：____

## 申请流程（通用版）

1. 选卡与确认条款（年费/欢迎礼/门槛）
2. 准备材料并提交（线上/分行/推广渠道）
3. 审批与补件（可能电话核实）
4. 批核/拒绝（原因记录）
5. 收卡、开卡与设定（自动转账/还款日/额度管理）

## 提高通过率的做法（先占位）

- 控制短期内申请数量
- 解释资金来源/收入稳定性
- 提前准备可验证的地址与工作信息
