# 快速开始

## 目标

把选卡问题拆成三件事：

1. **你花钱最多的场景是什么**（餐饮/超市/网购/交通/旅游/外币）
2. **你想要的回报是什么**（现金回赠 / 里数 / 酒店积分 / 旅行权益）
3. **你能接受的成本与门槛是什么**（年费、最低年薪/资产、开户要求、刷卡门槛）

## 先做一份你的“用户画像”

把下面空格填上（后面所有章节都会用到）：

- 常住地：____（香港/内地/两地）
- 月均可刷卡金额：____ HKD
- 核心消费：____（例如：餐饮 40%，超市 20%，网购 20%，交通 10%，其他 10%）
- 外币/旅行频率：____（每年 ___ 次）
- 目标：____（例如：每年攒 ___ 里数；或回赠率稳定 > ___%）
- 能接受年费：____（0 / 可免 / 可付到 ___ HKD）

## 三条最重要的“选卡原则”

- **先拿欢迎礼，再谈长期**：欢迎礼通常决定第一年的回报率上限。
- **算清“真实回赠”**：把兑换损耗、上限、类别限制、外币手续费都算进去。
- **少而精**：主力卡 1-2 张 + 场景补充卡 1 张，通常比一堆卡更容易跑满回赠。

- 身份/地址证明：____
- 入息证明：____
- 银行往来/资产证明（如需要）：____

## 下一步阅读

- 去看：**选卡方法论**（把所有卡放进同一套比较框架）
- 如果你目标是里数：**里数与积分**
- 如果你总被拒：**风控/封卡/拒批应对**
